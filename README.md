## დავალება - 16. react-ის კომპონენტების გასტილვა

- განავრცე Todo - List აპლიკაცია და დაამატე კიდევ ერთი სვეტი “In Progress”
- ჯამში უნდა გქონდეს სამი სვეტი და მათ შორის უნდა შეგეძლოს task - ების მოძრაობა
- ასევე, სტილისტურად გააფორმე აპლიკაცია, რომ მიიღო მსგავსი სურათი

## Solution with style methods

- Form კომპონენტი გავსტილე css module მეთოდით - style.module.css ფაილით
- TodoList და Todo კომპონენტები - styled-components პაკეტით
- გამოყენებულია css ფაილით გასტილვა - index.css
- გამოყენებულია react in-line სტილებიც InProgressList და doneTasksList-ში
- დანარჩენი კომპონენტები TailwindCSS პაკეტის საშუალებით გავსტილე

## Functional solution

- დავალებაში ყოველ Todo, inProgress და Done კომპონენტებს დავუტოვე ღილაკები, რათა მომხმარებელმა ადვილად გადაადგილოს დავალებები
- ასევე დავტოვე ინფუთი, დავალებების დასამატებლად
- დავალებაში არ იყო წაშლის ღილაკზე მოთხოვნა, მაგრამ დავტოვე, უხდებასავით :)
